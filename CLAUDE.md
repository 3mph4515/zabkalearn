# Żabka Learn | Польский изнутри

## О проекте
Telegram-канал для изучения польского языка для русскоязычной аудитории (Украина, Беларусь, Россия).

## Автор
- Уровень польского: C1 (сдан экзамен)
- Локация: Польша
- Контент: курированный/переработанный, не авторский
- Фокус: автоматизация через ботов + ручной контент

## Позиционирование
- **Название:** Żabka Learn | Польский изнутри
- **Username:** @zabka_learn
- **Tone of Voice:** дружелюбный, молодёжный, без кринжа, сдержанно
- **УТП:** живу в Польше + C1 = реальный язык, не из учебников

## Описание канала
```
🐸 Польский язык — проще, чем кажется

Живу в Польше • C1
→ Слово дня из Żabki
→ Квизы и карточки
→ Разговорный > учебник

Открыто 24/7, как Żabka 🟢
```

### Шрифты
- Заголовки: Poppins Bold
- Текст: Inter

### Маскот
Милая минималистичная лягушка с польским румянцем (красные щёчки).

## Рубрики

### 🐸 Ciekawostka dnia
Интересные факты о польском языке. Формат:
- Карточка с необычным словом/фразой
- Квиз в Telegram (что это значит?)
- Пояснение после ответа

### Другие рубрики (планируемые)
- Słowo dnia — слово дня
- Podsluchano w Żabce — живой польский
- Błąd dnia — частые ошибки

## Целевая аудитория
1. **Трудовые мигранты** (35-40%) — нужен польский для работы
2. **Карта поляка** (30-35%) — подготовка к собеседованию
3. **Студенты** (15-20%) — поступление в польские вузы
4. **Беженцы** (10-15%) — вынужденное изучение
5. **IT/бизнес** (5-10%) — интеграция

## Монетизация (планы)
- Реклама в канале (с 5K подписчиков)
- Партнёрки (языковые школы, приложения)
- Платный контент (курсы, боты)

## Стратегия роста
- Комментинг в смежных каналах
- Взаимный пиар
- Статьи на VC.ru, Пикабу
- Виральный контент (мемы, квизы)
